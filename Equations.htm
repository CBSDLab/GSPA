This is the rate of building green space, which is represented as a regenerative growth process influenced by the amount of green space already available. For example, the number of trees and other plants in green space influence how fast they can be regenerated and propagated to create more green space.

"Community_Health_Improvement_Plan_(CHIP)"

MAX(Physical_Activity_Gap, 0)/Time_to_Close_Gap

1/Years

The community health improvement plan (CHIP) is a function of the size of the gap and the time to close the gap. That is, it represents how much of the gap must be closed each year in order to ideally close the gap within a given period of time. For example, if the gap is 10% and we want to close the gap in 5 years, we would expect to make progress at 2% per year for 5 years.

The max function is used to limit deimplementation when the prevalence of physical activity as exceeded the goal.

Net_Fractional_Rate_of_Change_In_Physical_Activity

Effect_of_Green_Space_Utilization_on_Change_in_Physical_Activity* IP4_Physical_Activity_Incentive*Initial_Fractional_Rate_of_Change_in_Physical_Activity+Implementing_CHIP

1/Years

The net fractional rate of change in physical activity, which is determined within the model as a function of the effect of green space. This can range from a negative fractional rate that represents deimplementation of physically active to zero which means no net change to a positive value that represents a net fractional rate of implementing physical activity.

In this formulation, implementation and deimplementation of physical activity are assumed to be symmetric, i.e., the factors that contribute to implementation are the same as those that contribute to deimplementation.

Since a model assumption is that it starts in a dynamic equilibrium, the initial effect of green space utilization is zero. As other initiatives are implemented that put the model into a disequilibrium, the net fractional rate is modified as a multiplier by implementing community health improvement plan (CHIP) and physical activity incentives.

Note that unlike the Bass diffusion model, incentives are treated as a multiplier and accelerate the direction of change as opposed to adjusting the fractional rate as an additive factor. That is, this does not have an effect when the system is in a dynamic equilibrium. This is something that one might want to reconsider and experiment with to see what impact a multiplicative versus additive formulation has on the model.

However, implementing the community health improvement plan (CHIP) is an additive effect, meaning it has an effect on the net fractional rate of change even when the system is in dynamic equilibrium.

SW_Stacked_Effect_Sizes

0

Dimensionless

This switch determines whether the overall effect size (ES) is allocated to each intervention or evenly distributed across active interventions.

A value of 1 stacks the ES such that if the Overall ES is 0.5, one active intervention will have a value of 0.5 and two active interventions will each have 0.5. However, if the value is 0 or anything else other than 1, the overall ES is evenly distributed. So one active intervention will have an ES of 0.5, two active interventions will have an ES of 0.25, three active interventions will each have an ES of Overall ES/3 or 0.5/3 = ~0.167, and so on.
